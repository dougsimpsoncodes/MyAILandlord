MyAI Landlord Design System

Unified design language for consistent user experience across all screens

Design Principles

1
Simplicity First

Remove redundant elements. One way to do each action.

2
Consistent Navigation

Same header pattern on every screen. Global bottom nav.

3
Clear Hierarchy

Primary actions are obvious. Secondary actions are subtle.

4
Mobile-First

Designed for thumb reach. Large tap targets. Clear spacing.

Current Issues to Fix

!
No Global Navigation

Each screen has different nav. Users get lost.

!
Inconsistent Headers

Some have back buttons, some don't. Title placement varies.

!
Redundant Add Buttons

Multiple ways to do same action (e.g., 3 "Add Property" buttons).

!
Duplicate Screen Patterns

Login/SignUp are 95% identical. Home screens duplicate code.
