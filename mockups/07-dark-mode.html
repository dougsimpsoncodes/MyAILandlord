Enhancement #7: Dark Mode Support

Reduce Eye Strain & Battery Usage

WHY THIS MATTERS: Dark mode is expected by modern app users. It reduces eye strain in low-light environments, saves battery on OLED screens, and is often preferred by users who spend significant time in apps. iOS and Android both support system-wide dark mode that apps should respect.

IMPLEMENTATION APPROACH:
  • useColorScheme hook: Detect system preference with React Native's built-in hook
  • ThemeContext: Create a context to provide colors throughout the app
  • Semantic colors: Use names like "background" not "white" so they adapt
  • Manual toggle: Allow users to override system preference in settings
  • Persist preference: Save user's choice in AsyncStorage
