# Test Fixtures

This directory contains test assets used in Playwright E2E tests.

## Test Images

The following images are used for testing photo upload functionality:

- `property-exterior-1.jpg` - Front view of property
- `property-exterior-2.jpg` - Side view of property
- `property-exterior-3.jpg` - Back view of property
- `room-living-room.jpg` - Living room interior
- `room-kitchen.jpg` - Kitchen interior
- `asset-refrigerator.jpg` - Refrigerator asset photo

## Usage

These fixtures are referenced in Playwright tests for:
- Property photo upload testing
- Room photography testing
- Asset photo upload testing
- File validation testing

## Image Requirements

All test images should be:
- Valid JPEG format
- Between 100KB - 5MB in size
- Minimum 800x600 resolution
- Maximum 4096x4096 resolution
